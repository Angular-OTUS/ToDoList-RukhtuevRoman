# ToDo List

Простое приложение для управления задачами, созданное на Angular.

## Функциональность

- ✅ Добавление новых задач
- ✅ Отметка задач как выполненных
- ✅ Удаление задач

## Технологии

- **Angular** 20+
- **TypeScript**
- **SCSS**

## Установка и запуск

Установка зависимостей

```bash
npm install
```

Запуск frontend

```bash
npm run start
```

Запуск json-server

```bash
npm run start:api
```

Запуск frontend и json-server одной командой

```bash
npm run dev
```

Проект в браузере

```bash
http://localhost:4200
```
