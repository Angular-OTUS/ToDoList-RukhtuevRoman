# ToDo List

Простое приложение для управления задачами, созданное на Angular.

## Функциональность

- ✅ Добавление новых задач
- ✅ Отметка задач как выполненных
- ✅ Удаление задач

## Технологии

- **Angular** 20+
- **TypeScript**
- **SCSS**

## Установка и запуск

Установка зависимостей

```bash
npm install
```

Запуск проекта

```bash
ng serve
```

Проект в браузере

```bash
http://localhost:4200
```
